Como o Arena Virtual funciona

Uma aplicação de gerenciamento, criação, personalização e qualquer coisa sobre campeonatos. Essa api tem como o objetivo a facilidade e liberdade em criar qualquer campeonato e a interação entre os usuários que podem ser de diferentes modalidades, ou seja, campeonatos sobre qualquer tema, seja jogos, esportes, uma simples brincadeira, etc.
